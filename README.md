IoT Making for Django
====

AWSから取得したデータの見える化をDjangoと他ライブラリを使用して行った場合の実装集

## Description

### フロント

- Char.js、DataTablesで見える化

### サーバ

- Herokuを想定

### app

- api
  - django rest frameworkを使用してAWSからデータ取得
- iot_making_auth
  - Auth0から取得したトークンの管理、保持

## Requirement

- python3.6
